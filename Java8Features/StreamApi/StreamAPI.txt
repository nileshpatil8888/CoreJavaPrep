Stream API :
    It provides a functional and declarative way to process a collection of data.
    It supports operations for filtering, transforming, aggregating elements, making it easier to write clean and efficient code.

    Source: A collection or data source (e.g., List, Set, Array).
    Intermediate Operations: Transform the data (e.g., filter, map).
    Terminal Operations: Produce a result or a side effect (e.g., collect, forEach).
